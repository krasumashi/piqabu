Piqabu for iPhone

iPhone via SideStore

No App Store gatekeeper. Installs without asking Apple's permission.
Three steps, one week's maintenance.

  1. 01

    Install SideStore

    iPhone install is done through SideStore (free). Set it up once.

    Get SideStore  ↗
  2. 02

    Add Piqabu to SideStore

    Adds Piqabu as a source in SideStore, so you can install it and get updates from inside the app.

    Add to SideStore  →
    Deep link didn't fire? Paste this into SideStore manually.
    https://piqabu.live/apps.json
    Or grab the .ipa directly Download Piqabu.ipa  ↓ Safari saves the file, then open it in SideStore.
  3. 03

    Keep it active

    Apple limits free-installed apps to 7 days. Just open SideStore about once a week and it re-signs Piqabu automatically to keep it working.

Honest note

iPhone runs the full Piqabu app — rooms, Reveal/Peek, Whisper voice, Live Glass video. The Piqabu Keyboard is Android-only, and some device-level protections (screenshot blocking) aren't possible on iOS — that's Apple's sandbox, not a Piqabu limitation.
